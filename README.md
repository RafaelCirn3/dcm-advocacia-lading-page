# DCM Advocacia - Landing Page

Projeto criado para fins de pratica em front-end, usando Angular e componentes standalone.

## Como executar

```bash
npm install
npm start
```

Acesse `http://localhost:4200/` no navegador.

## Funcionalidades

- Landing page responsiva e moderna para escritorio de advocacia.
- Navegacao por ancoras com scroll suave.
- Carrossel de areas de atuacao sem bibliotecas externas.
- Botao de WhatsApp flutuante.
- Tema 2 (modo claro) com alternancia de paleta.
- Formulario de contato com envio simulado.
